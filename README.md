## How to compile NOTE: For windows recommend to use MSYS2 i.e. g++.exe (Rev5, Built by MSYS2 project) 16.1.0

- ```build_raylib.sh```

- ```make```

- The compiled binary will be in root directory it requires data/ and a testing project is in example/
