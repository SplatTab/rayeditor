**rayeditor is a work in progress editor using raylib and imgui as a backend**

## How to compile

### For windows recommend to use MSYS2 G++ "g++.exe (Rev5, Built by MSYS2 project) 16.1.0" is my version

- Run build_raylib.sh to automatically pull latest version of raylib and compile the correct dlls and have them copied to the correct locations(recommend)

- Run the make file

-The compiled binary will be in root directory it requires data/ and a testing project is in example/
